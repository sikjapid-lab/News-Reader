# پروژه کامل سایت خبری هوشمند جهان‌نما (GitHub Pages + Hugging Face AI)

این پروژه یک سامانه ۲۴ ساعته پایش، ترجمه و دسته‌بندی اخبار از ۲۰ خبرگزاری معتبر جهانی است.

## اجزای معماری پروژه
۱. **فرانت‌اند (GitHub Pages):** فایل `index.html` با رابط کاربری مدرن، راست‌چین، پشتیبانی از تم دارک/لایت، فیلترهای موضوعی و جغرافیایی و مدال نمایش جزییات خبر.
۲. **پردازشگر اخبار (Python):** فایل `process_news.py` که فیدهای RSS را دریافت، اخبار ۲۴ ساعت اخیر را فیلتر و جهت ترجمه به هوش مصنوعی می‌فرستد.
۳. **سرور هوش مصنوعی (Hugging Face Spaces):** فایل `app_hf.py` که مدل ترجمه انگلیسی به فارسی را میزبانی می‌کند.
۴. **اتوماسیون (GitHub Actions):** فایل `.github/workflows/update_news.yml` که هر ۳۰ دقیقه بیدار شده و اخبار جدید را بروزرسانی می‌کند.

## راهنمای راه اندازی:
۱. کدها را در یک مخزن جدید در گیت‌هاب آپلود کنید.
۲. در تنظیمات GitHub Repository به بخش **Pages** رفته و منبع را روی شاخه `main` قرار دهید.
۳. یک Space رایگان در Hugging Face ایجاد کرده و فایل `app_hf.py` را در آن قرار دهید.
۴. آدرس API و توکن Hugging Face را در بخش `Settings -> Secrets` گیت‌هاب وارد کنید.
